# On the Stability and Robustness of Vision Transformers for Neurodegenerative Disease Classification

This repository contains the implementation resources for the paper **"On the Stability and Robustness of Vision Transformers for Neurodegenerative Disease Classification"**, submitted to **MIDL 2026**.

## Code Release Status

**The full source code is currently being finalized for public release.**

We are committed to reproducibility. The complete codebase, including training scripts, model definitions, and stabilization protocols, will be made publicly available in this repository upon acceptance of the paper.

## Dependencies

*A full `requirements.txt` file will be provided in this repository.*

## Datasets

The study is based on the following cohorts: **ADNI**, **ALLFTD**, **NIFD**, and **NACC**.
Due to data usage agreements, we cannot share the raw data, but subject splits will be provided for reproducibility.
